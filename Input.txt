Задание.
Составить алгоритмы и программы с использованием текстовых файлов следующим образом:
1)	Создать исходный текстовый файл с помощью текстового редактора – файл Input.txt.
2)	Программа считывает текст из файл Input.txt, выполняет указанные действия и записывает полученный результат в файл Result.txt.
3)	Продублировать на экран выполняемые действия. 
4) Информация в файле и на экране должна быть понятной для пользователя. 
5)	Оформить отчет.
